## 🛠️ Troubleshooting

### The app does not open

Try these steps:

1. Right-click the app file.
2. Choose Run as administrator.
3. Check if Windows shows a security prompt.
4. If it still does not open, move the folder to your Desktop and try again.

### Windows says the file is blocked

1. Right-click the file.
2. Select Properties.
3. Check the Unblock box if you see it.
4. Click Apply.
5. Open the app again.

### The screen looks cut off

1. Maximize the window.
2. Change your display scale to 100% or 125%.
3. Make sure your screen resolution is at least 1366 x 768.

### The app closes right away

1. Make sure you extracted the full .zip file.
2. Check that all files stayed in the same folder.
3. Try downloading the file again.
